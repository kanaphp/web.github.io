บทคัดย่อ

โครงงานนี้เป็นโครงงานเกี่ยวกับเกมที่ถูกพัฒนาโดยใช้ภาษา C โดยมีจุดมุ่งหมายเพื่อ
สร้างความบันเทิงแก่ผู้เล่นอีกทั้งยังมีการนำความรู้ที่ได้มาปรับใช้ให้เข้ากับการพัฒนาด้วย

รายละเอียดโครงาน

โครงงาน Pick a Numba เป็นเกมที่ต้องใช้ผู้เล่น 2 คนโดยจะมีการสุ่มความสามารถ
ให้ใช้ได้ภายในเกม (ใน 1 เกมใช้ได้เพียง 1 ครั้งเท่านั้น) ได้แก่ เพิ่มแต้มฝั่งตัวเอง ลบแต้มฝั่งตรงข้าม
เพิ่มคะแนนการชนะแต่สุ่มคนที่จะได้หลังจากนั้นจะให้ผู้เล่นผลัดกันเลือกตำแหน่งเลข 0-9 (ดูว่าได้เลขอะไร)
คนละ 3 ครั้ง คนที่ได้แต้มมากกว่าจะชนะในรอบนั้นไป ซึ่งผู้เล่นที่ชนะ 2 ใน 3 รอบจะเป็นผู้ชนะเกมทันที

วิธีเล่น

1. หาเพื่อนและอุปกรณ์ที่ run ภาษา C ได้
2. เมื่อ run code จะให้ใส่ชื่อผู้เล่นทั้ง 2
3. ก่อนเริ่มเกมจะมีการสุ่มความสามารถเช่นเพิ่มแต้ม, ลบแต้ม หรือบวกจำนวนชนะ
4. เมื่อเกมเริ่มผู้เล่นจะผลัดกันเลือกเลขคนละ 3 ครั้งทั้งหมด 3 รอบ
5. คนที่ชนะ 2 ใน 3 รอบจะเป็นผู้ชนะ
